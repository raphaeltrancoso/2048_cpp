#####################################################################################
I/PRESENTATION
 Notre projet contient 3 jeux différences:
 - Taquin
 - 2048 avec deux versions supplémentaires
 - Sokoban (avec 5 niveaux)			        		                   
#####################################################################################
II/MANUEL
Pour jouer au jeu, suivez les instructions suivantes : 

1- Ouvrez un terminal sur le dossier du projet
2- Tapez la commande : make
3- Sur ce même terminale, tapez ensuite « ./[nom_du_jeu] » avec nom_du_jeu c'est le nom du jeu que vous voulez jouer(Taqui, game2048, game2048_2, game2048_3, sokoban) .  
4- Utilisez les "arrows keys" pour bouger les cases
5- Pour jouer avec un robot "stupid" utiliser le bouton 'r' pour son tour
6- Pour enlever le jeux, tapez: make clean
7- Jouez et amusez-vous bien !! ;)
